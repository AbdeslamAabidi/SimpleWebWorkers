SimpleWebWorkers 
================
Simple wrapper to create, start and stop web workers from an internal script or external javascript files.

## Versions
  v0.1: First Release

USAGE:
======

## Create Worker 1
```js
    var $_1 = $_w(function_1);
```

## Start Worker 1
```js
    $_1.start()
```

## Stop Worker 2
```js
    $_2.start()
```
